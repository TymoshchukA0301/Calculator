Опис інтерфейсу:
    Текст на кнопках розміщений по центру кнопки.
    Текст на екрані розміщений в правому нижньому кутку.

    Кнопки: '1' '2' '3' '4' '5' '6' '7' '8' '9' ',' '0' заповнюются кольором: HEX #424242.
    Кнопки: 'C' '+/–' '%' '!' '√' 'π' заповнюються кольором: HEX #616161.
    Кнопки: '+' '–' '×' '÷' '^' заповнюються кольором: HEX #ef6c00.

Функціонал:
    Вихідне положення Екрану:  пишеться 0.
    Набір числа:
        Користувач набирає число за допомогою кнопок: '1' '2' '3' '4' '5' '6' '7' '8' '9' '.' '0'.
        Кнопка '.' може бути натиснута тільки один раз під час набору числа.
        Якщо на початку набору числа натискається кнопка '0', то вона не відображається на екрані.
        Набране число відображається на Екрані.
    Виклик дії '+' '–' '×' '÷' '^':
        Після натискання кнопок: '+' '–' '×' '÷' '^', текст на екрані форматується в число і записується в пам'ять.
        Дія не відображається на екрані, але записується в пам'ять.
        Кнопка викликаної дії змінює свій фон так, щоб він виділявся.
        При наборі нового числа, попереднє число стирається.
    Функціонал кнопок '%' '+/–' 'C' '!' '√' 'π' '=':
        При натисканні кнопки '%', обчислюється процент набраного числа. Результат відображається на екрані.
        При натисканні кнопки '+/–', змінюється знак набраного числа на протилежний. Результат відображається на екрані.
        При натисканні кнопки 'C' один раз, стирається відображене на екрані число.
        При натисканні кнопки 'C' ще раз, вся збережена в пам'яті інформація стирається.
        При натисканні кнопки '!' обчислюється факторіал набраного числа. Результат відображається на екрані.
        При натисканні кнопки '√' обчислюється квадратний корень набраного числа. Результат відображається на екрані.
        При натисканні кнопки 'π' на екрані відображається число π - 3.14...
        При натисканні кнопки '=', відбувається обчислення: [Число A] - [Дія] - [Число B]. Результат відображається на екрані.
